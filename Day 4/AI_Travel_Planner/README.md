# AI Travel Planner (CrewAI Project)

## Overview
This is a multi-agent application built using CrewAI.

## What it does
The app helps users generate personalized travel plans instead of research reports.

## How it works
- Agent 1: Destination Finder → suggests places
- Agent 2: Itinerary Planner → creates schedule
- Agent 3: Guide Writer → generates detailed travel plan

## Workflow
User Input → Agents collaborate → Travel itinerary output

## Run the app
```bash
pip install -e .
python main.py
```

## Notes
Same architecture as research app, adapted for travel planning use case.
